# 中值定理

## 函数中值定理

条件： $f(x)$ 在区间 $[a,b]$ 上 `连续`

1. 最值定理 （有界 & 最值）

    $m \le f(x) \le M$, 

2. 介值定理

    若 $\mu \in [m, M]$, 则存在 $f(\xi) = \mu, \ \xi \in [a,b]$

3. 均值定理

    若 $a < x_1 < x_2 < \dots < x_n < b$, 则至少存在一个 $\xi \in [x_1, x_2], => f(\xi) = \frac{f(x_1) + f(x_2) + \dots + f(x_n)}{n}$

4. 零点定理

    若 $f(a)f(b)<0$, 则存在 $f(\xi) = 0, \ \xi \in [a,b]$
